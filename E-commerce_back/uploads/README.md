# Uploads 目录

这个目录用于存储用户上传的文件。

## 目录结构

```
uploads/
├── avatars/           # 用户头像文件
│   └── default-avatar.svg  # 默认头像
├── products/          # 商品图片（如需要可创建）
└── documents/         # 其他文档文件（如需要可创建）
```

## 文件说明

- `avatars/default-avatar.svg` - 用户注册时的默认头像
- 其他用户上传的文件将根据类型分类存储

## 注意事项

1. 确保服务器配置了静态文件服务，以便前端可以访问这些文件
2. 建议配置文件大小限制和文件类型验证
3. 定期清理无用的上传文件
